## Overview

The Fork offers three intensity levels that control how direct, provocative, and profane "Other You" will be in conversation. Each level maintains honesty but adjusts the delivery style.

<Warning>
All intensity levels allow profanity. None allow abuse, slurs, threats, or demeaning identity attacks.
</Warning>

## The Three Levels

### Mild

<Card title="MILD" icon="heart">
  **Supportive. Honest. Still has a pulse.**
</Card>

From `server.py:131-136`:

```python
if intensity == "mild":
    return (
        "MILD: supportive, reflective, grounded. Still honest. "
        "Use light profanity sparingly if it fits (e.g., 'damn', 'hell', 'bullshit'), "
        "but keep it caring. No cruelty."

```

**Characteristics:**
- Supportive, reflective, grounded
- Still honest — doesn't sugarcoat
- Light profanity sparingly (damn, hell, bullshit)
- Caring tone
- No cruelty

**Use this when:** You want honest reflection without the edge. You're exploring a decision but need a gentler approach.

## How Intensity Affects the Conversation

Intensity influences several aspects of the AI's behavior:

### 1. Profanity Mirroring

From `server.py:237-240`:

```python
style_bits.append(
    "Mirror the user's swearing level; in Savage/Brutal you can go one notch dirtier."
)
```

The AI mirrors your language style but can escalate slightly in higher intensities.

### 2. Paragraph Structure

From `server.py:243-246`:

```python
if intensity in ("savage", "brutal"):
    style_bits.append("No long paragraphs. 1–2 sentences per paragraph max.")
else:
    style_bits.append("Keep paragraphs tight. Don't ramble.")
```

Higher intensities favor shorter, punchier paragraphs.

### 3. Base Persona

All intensities share the same base voice (from `server.py:267-272`):

<Info>
"You have the vibe of a smartass biker with absolutely zero filter in a public setting. You're sharp, funny, and blunt. You say the quiet part out loud. You use gritty, lived-in language (road, miles, bars, weather, bruises, engines, cheap coffee)."
</Info>

```typescript
type Intensity = "mild" | "savage" | "brutal";
```

```python
Intensity = Literal["mild", "savage", "brutal"]
```

### Default Value

If not specified, the intensity defaults to **"mild"** (`server.py:102-104`).

## Changing Intensity Mid-Conversation

<Warning>
The Fork is a stateless chat. Each request includes the full conversation history + current intensity setting. You can change intensity between messages, and the AI will adjust its tone accordingly.
</Warning>

## Safety Note

Regardless of intensity level, the backend enforces safety checks:

- Self-harm detection (returns crisis resources)
- Hate speech detection (refuses and redirects)
- System prompt prohibits slurs, threats, and identity attacks

See [Safety & Ethics](/features/safety-ethics) for full details.
