<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get The Fork running locally in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone The Fork from GitHub and navigate to the project directory.

      ```bash
      git clone https://github.com/CassieMarie0728/the-fork.git
      cd the-fork
      ```
    </Step>

    <Step title="Set up your environment">
      Copy the environment template and add your Emergent LLM API key.

      ```bash
      # Backend configuration
      cd backend
      cp .env.example .env
      # Edit .env and set EMERGENT_LLM_KEY
      ```

      <Note>
        You need an `EMERGENT_LLM_KEY` to enable chat functionality. The application won't work without it.
      </Note>
    </Step>

    <Step title="Run with Docker Compose">
      Launch the full stack with a single command.

      ```bash
      docker-compose up --build
      ```

      Access the application at `http://localhost:3000` and the API at `http://localhost:8000`.
    </Step>

    <Step title="Create your first conversation">
      Open the app, enter a fork statement describing the path you didn't take, select an intensity level, and start talking to your alternate self.

      <Info>
        All conversations are stateless and ephemeral. Nothing is persisted beyond the current session.
      </Info>
    </Step>
  </Steps>
</div>
